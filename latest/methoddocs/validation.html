Validation

Functions for validating EIP-712 message structure (required fields, etc.).

eip712.validation.used_header_fields(EIP712Domain_data)

Returns only the EIP712_DOMAIN_FIELDS from the given domain data.

eip712.validation.validate_EIP712Domain_schema(structured_data)

Verifies that the given structured_data contains a valid EIP-712 domain schema. Raises eth_utils.ValidationError otherwise.

eip712.validation.validate_field_declared_only_once_in_struct(field_name, struct_data, struct_name)

Validates that the given field_name only appears once in struct_data. Raises eth_utils.ValidationError otherwise.

eip712.validation.validate_has_attribute(attr_name, dict_data)

Validates whether dict_data contains attr_name, raising eth_utils.ValidationError if it does not.

eip712.validation.validate_primaryType_attribute(structured_data)

Verifies that the given structured_data contains a valid primaryType definition. Raises eth_utils.ValidationError otherwise.

eip712.validation.validate_structured_data(structured_data)

Top-level validator that verifies structured_data is a valid according to the EIP-712 spec. Raises eth_utils.ValidationError otherwise.

eip712.validation.validate_types_attribute(structured_data)

Validates the existence of the types field, its contents, and whether identifer and type names are valid. Raises eth_utils.ValidationError otherwise.
